# LLM Inference Performance Model

This repository, "LLM-Inference-Performance-Model", is dedicated to exploring and modeling the performance characteristics of Large Language Model (LLM) inference. It aims to provide insights and tools for understanding, predicting, and optimizing the computational resources required for running LLMs efficiently.

For more details, visit the [project homepage](https://anchieh.github.io/LLM-Inference-Performance-Model/).
